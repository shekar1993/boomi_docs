# Adding a Domain

To add a new domain: 

## Procedure

1. Go to **Manage > Domains**. 

2. Click on **New Domain**.

   The **New Domain** pop-up window is displayed. 

3. Enter Domain suffix in the **Domain Suffix** field. 

4. Click **Save and close**.

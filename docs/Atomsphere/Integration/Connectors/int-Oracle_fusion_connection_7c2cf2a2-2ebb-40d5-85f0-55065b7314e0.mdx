The Oracle Fusion connection also known as Oracle ERP Cloud connection represents a single account using login credentials. If you have multiple systems, you need a separate connection for each.

## Connection tab

The Oracle Fusion connection requires the following connection properties:

**Fusion Endpoint URL** - 
Refers to the host URL of the Oracle Fusion.

**Security Type** - 
Refers to the BASIC authentication protocol.

**User Name** - 
Refers to the Oracle Fusion user.

**Password** - 
Refers to the Oracle Fusion user password.

:::note

Whenever you modify the connection details **\(Fusion Endpoint URL/User Name/Password\)**, make sure to reimport the profile for the selected operation.

:::

## Connection Types \(BASIC\)

The Oracle Fusion connector supports the following protocol and authentication type:

-   Protocol
    -   HTTPS
-   Authentication Type
    -   BASIC

# Interactive Documentation Portal Page Settings

<head>
  <meta name="guidename" content="API Management"/>
  <meta name="context" content="GUID-c54ff60e-f677-476a-82a2-0db64a45e126"/>
</head>

The **Interactive Documentation** section of the portal can be customized by adding a title and a description. The title and description are displayed on the **Interactive Documentation** portal page.

All available access control-provided Interactive API Documentation items are displayed by their **"Title"** in the drop-down list.

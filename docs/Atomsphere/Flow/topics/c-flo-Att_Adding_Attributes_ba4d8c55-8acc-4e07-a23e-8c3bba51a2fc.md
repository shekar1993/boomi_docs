# Adding attributes to a component

<head>
  <meta name="guidename" content="Flow"/>
  <meta name="context" content="GUID-ba4d8c55-8acc-4e07-a23e-8c3bba51a2fc"/>
</head>


You can add supported attributes to a component in the page builder.

1. Select a component in the page builder.
2. Click **Add Attribute** in the **Attributes** section of the component configuration panel.
3. Configure the attribute:
    - **Key**: Select or enter a supported attribute key.

    - **Value**: Enter a supported value for the attribute. For example, many attributes require a 'true' or 'false' value. Values are case-sensitive and should be entered exactly as shown in the attributes reference.

4. The new attribute is shown in the **Attributes** section, where it can be edited or deleted as required.
5. Click **Add Attribute** to add additional attributes, or **Save** the page as normal.

The Aible AI – Partner connection represents credentials to connect multiple Aible tenants.

## Connection tab 

The Aible AI - Partner connector using the following information to connect to Aible:



**URL** - 
The Aible API endpoint. The default value is `https://api.iamaible.com` and does not normally need adjusting.

**USERNAME** - 
The username to log in to Aible. The user may have permissions to access projects across multiple tenants; a specific tenant is configured in the Operation where required.

**PASSWORD** - 
The password for the Aible user.

# Deleting Filters

<head>
  <meta name="guidename" content="API Management"/>
  <meta name="context" content="GUID-6dbe2d56-93dc-4a6a-b2b7-eb67a1907460"/>

Filters can be deleted. Filters once deleted cannot be retrieved. 

## Procedure

To delete a filter:

1. Click the filter. 

2. Click **Delete**.

The Boomi API Management API exposes several object types on which you can perform specific operations by issuing API calls. These operations are limited to ``GET`` (ordinary and bulk), ``QUERY``, ``CREATE``, ``UPDATE``, ``DELETE``, and ``EXECUTE``.
